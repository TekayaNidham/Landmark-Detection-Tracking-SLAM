# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:
- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the community

Examples of unacceptable behavior include:
- Harassment and personal attacks
- Trolling or insulting comments
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior and may remove, edit, or reject comments, commits, code, or other contributions that are not aligned with this Code of Conduct.

## Scope

This Code of Conduct applies within project spaces and in public spaces when an individual is representing the project.

## Reporting

Report unacceptable behavior to maintainers via private communication channels.
