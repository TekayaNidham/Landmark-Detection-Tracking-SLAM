# Security Policy

## Supported Versions

This project currently supports the default branch for security updates.

## Reporting a Vulnerability

Please do not open a public issue for suspected vulnerabilities.

Instead:
- Open a private security advisory in GitHub Security tab, or
- Contact the maintainers directly with reproduction steps.

Include:
- Affected files and function names
- Impact assessment
- Minimal reproduction details
- Suggested mitigation if available

We will acknowledge reports as quickly as possible and coordinate disclosure.
